This project is a designed only, never implemented, distributed social network without any
centralized node that turns theoretically impossible to be censored by anyone (even by the
developers).

The idea is based on signatures and the fact that you can store posts/userdata on any machine
without need to trust it to ensure that the data has not modified.

It's also based on a trust level system that automatically makes malicious servers be ignored
by clients and other nodes.

The full design doc can be read here: <https://docs.google.com/document/d/1Uih5z9-VHgBjntDHRN5_6zydmxjyjTozokD3DHWGyOs/edit>
